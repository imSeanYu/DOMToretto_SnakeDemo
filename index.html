How does it work?
    Each square is a "li" DOM Node element inside the DOMNodeCollection object, captured by DOMToretto's $l('li').
    The snake, apple, and blank "li" elements are rendered via setInterval(), by traversing through each() DOM Node element and conditionally calling addClass() / removeClass() functions to manipulate the DOM.
    attr() function is used to either fetch or assign the position value held by the DOM, and to consequently assign the green (snake) or the red (apple) CSS class attribute.
    Lastly, the snake object utilizes a Queue Data Structure.
